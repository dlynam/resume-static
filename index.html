Food Delivery City

http://fooddeliverycity.com

I built Food Delivery City for a startup. It allows restaurants to easily add a menu to their existing website with just one line of javascript. This site is built in Rails 3.1 and uses jQuery templates to render the menu. I built everything from the ground up, including the design, JSON API, order faxes, automated phone confirmation, and online payment.

  • Built restaurant menu renderer that uses just one line of js using jQuery Templates.
  • Built a fast menu api using JSON.
  • Built an automated phone call system using Asterisk and Adhearsion for order confirmation calls.
  • ActiveMerchant integration with Authorize.net for online payments.
  • eFax integration for outgoing order faxes.
  • Front end design / UI and CSS.

Rails 3.1 Ruby 1.9.2 MySQL jQuery jQuery Templates CoffeeScript JSON Adhearsion Asterisk eFax CSS

Flashcard Stash

http://flashcardstash.com

Flashcard Stash is a personal project of mine. It started as an online dictionary that allowed one to save words and has since grown into a flashcard based study tool. It has over 50,000 users and has been a great learning experience.

  • Implemented a dictionary api by parsing definitions from Wiktionary that uses Tokyo Cabinet.
  • Built a scraper to find appropriate sentence examples from online news sources (New York Times, Time magazine, The New Yorker, Wired Magazine) for every word.
  • Created an automated multiple choice and fill in the blank quiz that utilizes spaced repetition for more efficient learning.
  • Implemented a stable backend on a Linux host using Nginx, Memchached, and Thin Servers to scale to over 50,000 users.
  • High Score lists for games using Redis.
  • Facebook SDK integration.
  • Implemented a CSS generator that allows for customizable themes and backgrounds.

Rails 2.3.12 Ruby 1.8.7 MySQL Prototype Tokyo Cabinet Memchached Redis Amazon S3 Facebook SDK CSS
